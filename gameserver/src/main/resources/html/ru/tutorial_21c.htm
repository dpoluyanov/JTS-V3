[Смена класса Светлый Мистик]:
Достигший 20-го уровня Светлый Мистик может выбрать один из двух классов: Светлый Маг или Оракул Евы. Получив класс, персонаж получает доступ к большему разнообразию специальных умений. Персонаж 18-го уровня может выполнить квест на получение класса. Для того чтобы стать Светлым магом, обратитесь к Розалии в Эльфийской Деревне, а для становления Оракулом Евы - к Мануэлю в деревне замка Глудио. Хорошенько подумав, выберите себе подходящий класс.
