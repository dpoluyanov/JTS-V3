Изучение умений

Поздравляем! Вы наконец-то достигли 5-го уровня.
Теперь Вы можете изучить первое умение. Если хотите освоить его, Вам необходимо встретить Владыку в Деревне. Если пойдете туда, куда укажет стрелка над Вашей головой, то встретите Владыку.
Для того чтобы изучить Умение, необходимо потратить определенное количество Очков Умений (SP). Во время охоты SP увеличивается при убийстве каждого монстра. Если у Вас не хватает SP для изучения умения, Вам надо увеличить количество SP, охотясь на монстров.
Чтобы проверить, сколько SP осталось, откройте Окно Персонажа. Для этого надо щелкнуть по иконке с изображением лица в правом нижнем углу экрана или нажать клавиши ALT+T.
