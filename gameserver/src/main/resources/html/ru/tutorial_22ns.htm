Изменение класса Темного Рыцаря:
Чтобы стать Темным Танцором, игрок должен успешно пройти три квеста. Испытание Верности может проходить только персонаж, достигший уровня уровня 35 и выше. Этот квест начинается со встречи с Великим Мастером Ханнавальтом Города Гиран. Испытание Судьбы может проходить только персонаж, достигший уровня уровня 37 и выше. Этот квест начинается со встречи с Магистром Кайрой из Города Гиран. Наконец, Испытание Колдуна может проходить только персонаж, достигший уровня 39 и выше. Этот квест начинается со встречи с Оримом Тенью у Академии Хардина.
