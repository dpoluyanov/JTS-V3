Смена профессии Собирателя:
Собиратель, достигший 40-го уровня, может во второй раз сменить профессию и стать Охотником за Наградой. Для этого игрок должен успешно пройти три квеста. Испытание Умения может проходить только персонаж, достигший 35-го уровня. Этот квест начинается со встречи со Смотрителем Склада Вальконом из Гирана. Испытание Успеха может проходить только персонаж, достигший 37-го уровня. Это задание начинается со встречи со Смотрителем Склада Парманом из Гирана. Наконец, Испытание Ищущего может проходить только персонаж, достигший 39-го уровня. Этот квест начинается со встречи с Мастером Лютером из Деревни Охотников.
