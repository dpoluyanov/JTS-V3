Разведчик - смена профессии:
Чтобы стать Серебряным Рейнджером, Разведчик должен успешно пройти три квеста. Испытание искателя может выполнять только персонаж, достигший 35-го уровня. Этот квест начинается со встречи с Мастером Дафнером из Гирана. Испытание Жизни доступно только по достижении 37-го уровня. Поговорите с Мастером Кардиен из Диона. Наконец, Испытание меткости может проходить только на 39-м уровне или выше. Найдите Главу Гильдии Охотников Бернарда из Деревни Охотников.
