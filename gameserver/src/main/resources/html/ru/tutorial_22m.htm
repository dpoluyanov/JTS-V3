Смена профессии Ремесленника:
Ремесленник, достигший 40-го уровня, может во второй раз сменить профессию и стать Кузнецом. Для этого игрок должен успешно пройти три квеста. Испытание Умения может проходить только персонаж, достигший 35-го уровня. Этот квест начинается со встречи со Смотрителем Склада Вальконом из Гирана. Испытание Успеха может проходить только персонаж, достигший 37-го уровня. Вам нужно встретиться со Смотрителем Склада Парманом из Гирана. Наконец, Испытание Мастера может проходить только игрок, достигший 39-го уровня. Этот квест начинается со встречи с Локрином из Гильдии Железных Врат в Деревне Гномов.
