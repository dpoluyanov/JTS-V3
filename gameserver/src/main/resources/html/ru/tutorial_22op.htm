Смена профессии Ассасина:
Чтобы стать Призрачным Рейнджером, игрок должен успешно выполнить три квеста. Испытание Искателя может пройти только персонаж 35-го уровня и выше. Этот квест начинается со встречи с Мастером Дафнером в Гиране. Испытание Судьбы доступно только персонажу 37-го уровня и выше. Для начала необходимо встретиться с Магистром Кайрой в Гиране. Наконец, Испытание Стрельца может пройти только персонаж 39-го уровня и выше. Этот квест начинается со встречи с Бернардом, Главой Гильдии Охотников.
