Чтобы стать Епископом, Клерик должен успешно пройти три квеста. Испытание Паломника может проходить только игрок, достигший уровня 35 и выше. Этот квест начинается со встречи с Отшельником Сантьяго в сельском доме неподалеку от Бараков Орков в Глудио. Испытание Доверия может проходить только игрок, достигший уровня 37 и выше. Этот квест начинается со встречи с Верховным Жрецом Холлинтом из Города Орен. Наконец, Испытание Целителя может проходить только игрок, достигший уровня 39 и выше. Этот квест начинается со встречи со Жрецом Банделлосом из Города Гиран.
