Маг - смена профессии:
Чтобы стать Властителем Огня, необходимо успешно пройти три квеста. Испытание Мудрости доступно только по достижении 35-го уровня. Этот квест начинается со встречи с Магистром Мириен из Диона. Испытание Доверия может проходить только игрок, получивший 37-й уровень. Для начала поговорите с Верховным Жрецом Холлинтом из Орена. Испытание мага можно взять начиная с 39-го уровня. Найдите Барда Рукала в Дионе.
