Изменение класса Монаха:
Монах, достигший уровня 40, может во второй раз изменить класс и стать Отшельником. Чтобы стать Отшельником, игрок должен успешно пройти три квеста. Испытание Храбрости может проходить только персонаж, достигший уровня 35 и выше. Этот квест начинается со встречи с Кэшем из Города Дион. Испытание Славы может проходить только персонаж, достигший уровня 37 и выше. Этот квест начинается со встречи с Префектом Вокианом из Города Гиран. Наконец, Испытание Силы может проходить только персонаж, достигший уровня 39 и выше. Этот квест начинается со встречи с Дуэлистом Каеном из Города Орен.
