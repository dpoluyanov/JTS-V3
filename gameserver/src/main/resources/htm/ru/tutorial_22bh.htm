Чтобы стать Стрелком, Разбойник должен успешно пройти три квеста. Испытание Пытливости может проходить только игрок, достигший 35-ого уровня и выше. Этот квест начинается с визита к Мастеру Дафнеру из Города Гиран. Испытание Доверия может проходить только игрок, достигший 37-ого уровня и выше. Этот квест начинается со встречи с Верховным Жрецом Холлинтом из Города Орен. Наконец, Испытание Стрельца может проходить только игрок, достигший 39-ого уровня и выше. Этот квест начинается со встречи с Бернардом, Главой Гильдии Охотников из Деревни Охотников.
