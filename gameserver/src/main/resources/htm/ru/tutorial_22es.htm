Светлый Рыцарь - смена профессии:
Чтобы стать Менестрелем, Светлый Рыцарь должен успешно пройти три квеста. Испытание храбрости доступно, начиная с 35-го уровня. Этот квест начинается со встречи с Кэшем в Дионе. Испытание Жизни может пройти только персонаж 37-го уровня и выше. Для начала необходимо поговорить с Мастером Кардиен из Диона. Наконец, Испытание силы можно проходить с 39-го уровня и выше. Найдите в Орене Дуэлиста Каена.
