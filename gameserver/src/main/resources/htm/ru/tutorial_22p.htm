[Смена профессии Темного Волшебника]:
Достигнув 40-го уровня, Темный Волшебник может сменить профессию на Заклинателя Ветра или Последователя Тьмы. Для того чтобы сменить 2-ю профессию, Вам надо выполнить 3 квеста, первый из которых можно взять на 35-м уровне. Для более подробной информации щелкните по названию класса.
